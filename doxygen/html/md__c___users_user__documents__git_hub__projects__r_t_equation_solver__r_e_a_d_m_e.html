Quad solver project
RT_equation_solver

How to use

Run via console Enter compile.bat and quad_solver.exe

compile.bat
build\quad_solver.exe

If you want, you can add flags to the line above, which will be discussed below. Enter three numbers, separated by spaces. These three numbers are the coefficients of the quadratic equation, starting with the highest coefficient. Get the correct answer and realize that real men solve the square not by Vieta, but by RT_equation_solver
